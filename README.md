# Tic Tac Toe

A clean, browser-based Tic Tac Toe game with a dark UI and smooth animations.

**Play it here:** [https://mayasuki.github.io/tic-tac-toe/](https://mayasuki.github.io/tic-tac-toe/)

## Features

- Two-player local gameplay (X vs O)
- Win detection with highlighted winning cells
- Confetti animation on win
- Draw detection
- Smooth animations (fade-in, pop-in on moves, pulse on win)
- Responsive dark gradient design

## How to Play

1. Open the link above in any browser
2. Player X goes first — click any cell to place your mark
3. Players alternate turns until someone wins or the board is full
4. Click **Restart** to start a new game

## Tech Stack

- Plain HTML, CSS, and JavaScript — no frameworks, no dependencies
- Single file (`index.html`)

## Run Locally

Just open `index.html` in your browser — no server needed.
